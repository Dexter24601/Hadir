About


Create a Class

By clicking on the “Create button” the system will generate a QR code for the form and display it on the projector so the whole class can access it, each attendee will enter their name, ID, and 3 portrait photos of their face in each direction(front, left-side, right side) and submits the form, the system will then store their data in the database, each attendee will have his own profile in the system. This process will be done only once for each class/course.

Take Attendance

After creating a class, you can simply click on the created class and insert an image of the class in order to register the class attendance. This process will be done every day by the user. Through Face Recognition Technology. A copy of each image will be sent to the database for updating the dataset

View Attendence

You can view the attendance data via a dashboard to manage and view the overall attendance over the class. By selecting a course in the dashboard, the user can view the absence rate of students in that course. With the dashboard, the user experience is improved in terms of visibility and performance!
