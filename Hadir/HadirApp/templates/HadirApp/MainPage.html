Take Attendance


After creating a class, you can simply click on the created class and insert an image of the class in order to register the class attendance. This process will be done every day by the user. Through Face Recognition Technology. A copy of each image will be sent to the database for updating the dataset, and another copy will be sent to the server (engine) where it will be processed by the following operations: Face Detection The first step in the image processing operation is to detect human faces in the given image. Face Alignment Face images are aligned using landmarks before they are sent to the face recognition process. In addition to the bounding box and confidence score, it also outputs a five-point landmark. Feature Extraction feature extraction is the key to constructing an efficient predictive model.By the help of PCA algorithm (Principles Component Analysis) that extracts the mean face and their feature vectors and concatenates them as the feature vector for face recognition. Feature matching After extracting features and the mean face, Return Results Results will be returned back to the website as a grid whether a match was found or not, in case of a match, the names of the recognized faces will appear in green as present, the unrecognized faces will appear in grey as unknown, the names that have not found a match will appear in red and be considered absent.

View Attendence


You can view the attendance data via a dashboard to manage and view the overall attendance over the class. By selecting a course in the dashboard, the user can view the absence rate of students in that course. With the dashboard, the user experience is improved in terms of visibility and performance!

Create a Class


By clicking on the “Create button” the system will generate a QR code for the form and display it on the projector so the whole class can access it, each attendee will enter their name, ID, and 3 portrait photos of their face in each direction(front, left-side, right side) and submits the form, the system will then store their data in the database, each attendee will have his own row in the DB table and the website table. This process will be done only once for each class/course.
